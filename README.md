# Full Stack Blog Application (MERN)

A full-featured blog application built using the MERN stack with modern best practices like RTK Query, pagination, filtering, and authentication.

## Tech Stack

* Frontend: React, React Router, React Hook Form, Redux Toolkit, RTK Query
* Backend: Node.js, Express.js
* Database: MongoDB

## Features

* User authentication (login system)
* Create, update, delete blog posts
* Like and comment on posts (authenticated users only)
* Search functionality
* Category-based filtering
* Pagination and sorting
* RESTful API design

## Highlights

* Optimized API calls using RTK Query caching
* Clean and scalable backend structure
* Efficient state management using Redux Toolkit
* Real-world feature implementation

## Installation

### 1. Clone the repository

```
git clone https://github.com/munnadotin/blog-app
```

### 2. Install dependencies

```
cd frontend
npm install

cd backend
npm install
```

### 3. Run the project

```
npm run dev
```
